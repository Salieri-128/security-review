# Security Review

This repository documents my journey of learning smart contract security review through
**Cyfrin Updraft**.

It is intended to collect my audit practice, review notes, vulnerability writeups, and
learning summaries as I build a more systematic understanding of Web3 security.

## Goals

- Build a solid foundation in smart contract auditing.
- Practice identifying, explaining, and validating vulnerabilities.
- Keep a clear record of audit findings, notes, and lessons learned.
- Develop repeatable review habits for Solidity codebases.

## Contents

At this stage, the repository is a learning workspace. As it grows, it may include:

- Audit reports and practice findings
- Notes from Cyfrin Updraft lessons
- Vulnerability case studies
- Review checklists and methodology notes

## Status

This is an active learning repository. The structure and content will evolve as I
complete more review exercises and organize my notes.
